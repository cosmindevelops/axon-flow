# Axon Flow Development Setup Guide

Welcome to Axon Flow! This guide will help you set up your development environment.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Docker Desktop** (v24.0.0+) - [Download](https://www.docker.com/products/docker-desktop)
- **Node.js** (v20.18.0+) - [Download](https://nodejs.org)
- **Git** - [Download](https://git-scm.com)
- **Visual Studio Code** (recommended) - [Download](https://code.visualstudio.com)

## Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-org/axon-flow.git
   cd axon-flow
   ```
